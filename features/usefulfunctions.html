There are many built-in functions in LESS. Here are several examples.
@list: red, green, blue, orange; .body{ list-length: length(@list); background: extract(@list, 3); }
Length returns the number of elements in a value list. Extract returns the value at a specified position in a list.
.mixin(red) {background: red} .mixin(gray) {background: gray} .mixin(@col) when (default()) {background: @col} .body:before{ .mixin(red); } .body:after{ .mixin(grey); } .body{ .mixin(orange); }
Defaultavailable only inside guard conditions and returns true only if no other mixin matches, false otherwise.
.mixin(@property)when(isnumber(@property)){ height: @property; border-radius: @property/10; } .mixin(@property)when(iscolor(@property)){ background: @property; } .arms:before{ .mixin(150px); } .arms:after{ .mixin(orange); }
Isnumber returns true if a value is a number, false otherwise. Iscolor returns true if a value is a color, false otherwise.
@color: orange; .body{ background: @color; &:hover{ background: darken(@color, 20%); } } .arms:before, .arms:after{ background: lighten(@color, 30%); } .body:before, .body:after{ background: fadeout(@color, 30%); }
LESS provides several functions that we can use to manipulate colors: lighten(), darken(), saturate(), desaturate(), fadein(), fadeout(), fade(), spin(), mix().
@import "css/elements.less"; .arms:before{ .gradient(#6CB1EE, #4F85B2, #BCE6FF); } .arms:after{ .inner-shadow(2px, 5px, 10px, 0.5); }
Like standard CSS LESS allows @import rules files. But Less doesn't care where you put @import statements.
