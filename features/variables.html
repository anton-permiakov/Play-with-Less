Say "No" to the values repeated dozens of times across your stylesheet! Make your code easier with
LESS variables
@color: red; .body{ background: @color; }
The variables will allow us to store a constant value that later can be reused in the entire stylesheet.
@color: red; .body{ @color: blue; background: @color; &:before{ //is equal to .body:before background: @color; } } .arms:before{ background: @color; }
Variables will be inherited first from the local scope, and when it is not available locally it will search up a wider scope.
//Selectors @left-hand: arms:before; @right-hand: arms:after; //Properties @bg: background; .@{left-hand}{ @{bg}: yellow; } .@{right-hand}{ @{bg}: orange; }
Variables can also be used in other places as well, such as selector names and property names
@images: "images"; .body{ background: url("@{images}/dots.png") repeat #777; }
Variables can also be used in other places as well, such as URLs and @import statements
@border: 10px; @border-type: solid gray; .arms:before{ border-top: @border * 2 @border-type; border-right: @border - 5 @border-type; border-bottom: @border + 5 @border-type; border-left: @border - 2 @border-type }
We can also do something like addition, subtraction, multiplication and division to number, color and variable in the style sheet.
